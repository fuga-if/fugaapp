## 召喚獣

---

### ゼウス (Zeus)

> *最高神。ゲーム中最強の召喚獣。*

| 項目 | 値 |
|------|-----|
| **タイプ** | 召喚獣 |
| **ダメージ** | 4（実質即死） |
| **色** | #FFD700（金） |
| **カテゴリ** | 基本 |
| **クラス** | `ZeusCard extends SummonCard` |
| **ファイル** | `lib/theomachia/cards/zeus.ts` |

#### 効果
- ターン開始時に**4ダメージ**（HP4なので実質即死）
- **直接プレイ不可** — 「降臨」カード経由でのみ召喚できる
- メデューサの反射を**無視**する
- GUARDでしか防げない

#### 相互作用
| 対象 | 結果 |
|------|------|
| メデューサ（反射） | ❌ 無視（ゼウスは反射されない） |
| GUARD（打ち消し） | ✅ 唯一の防御手段 |
| 加護 | ❌ 無視（ゼウスは加護を貫通） |
| 奈落送り | ✅ 場から除去可能 |
| 均衡 | ✅ 場から除去可能 |
| 変身の秘術 | ✅ エリスと入れ替わる |

#### 戦略メモ
- デッキの最強カード。「降臨」とセットで運用が基本
- 相手がゼウスを出したら、GUARDを温存しておく必要がある
- 「変身の秘術」で自分のエリスをゼウスに変えるコンボも強力

- 「変身の秘術」でゼウスに化けさせるのが最強コンボ

### メデューサ (Medusa)

> *ダメージ反射能力を持つ守備型召喚獣。*

| 項目 | 値 |
|------|-----|
| **タイプ** | 召喚獣 |
| **ダメージ** | 0 |
| **特殊能力** | ダメージ反射 |
| **色** | #2ECC71（緑） |
| **カテゴリ** | 基本 |
| **クラス** | `MedusaCard extends SummonCard` |
| **ファイル** | `lib/theomachia/cards/medusa.ts` |

#### 効果
- ダメージを与えない（攻撃力0）
- **ダメージ反射**: 自分が受けるスキルダメージを相手に跳ね返す
- ゼウスのダメージは反射**できない**

#### 反射判定表
| 攻撃元 | 反射 |
|--------|------|
| エリス | ✅ 1ダメージ反射 |
| 雷霆 | ✅ 2ダメージ反射 |
| ゼウス | ❌ 反射不可 |
| 貫通の矢 | ✅ 1ダメージ反射 |

#### 戦略メモ
- 相手がエリスを出してきたら、メデューサを出すと毎ターン逆にダメージを与えられる
- 雷霆を使う前に相手の場にメデューサがいないか確認
- 攻撃力0なので単体では火力不足

### 降臨 (Ascension)

> *手札の召喚獣を場に出す儀式。ゼウスを召喚する唯一の方法。*

| 項目 | 値 |
|------|-----|
| **タイプ** | 儀式 |
| **色** | #F39C12（オレンジ） |
| **カテゴリ** | 基本 |
| **クラス** | `AscensionCard extends SpellCard` |
| **ファイル** | `lib/theomachia/cards/ascension.ts` |

#### 効果
- 手札の召喚獣1体を場に出す
- **ゼウスを場に出す唯一の手段**
- ターゲット選択: 手札の召喚獣カード

#### 使用条件
- 手札に召喚獣カードが必要（なければ効果なし）

#### 戦略メモ
- ゼウス+降臨のコンボが最強ムーブ
- エリスやメデューサを出すのにも使える（直接プレイでもOKだが）
- 手札に召喚獣がない状態で使うと無駄撃ちになる

### 血の契約 (Blood Pact)

> *自傷＋大量ドロースキル。命を削ってカードを得る。*

| 項目 | 値 |
|------|-----|
| **タイプ** | スキル |
| **自傷ダメージ** | 1 |
| **ドロー** | 3枚 |
| **色** | #8E44AD（紫） |
| **カテゴリ** | **オプション** |
| **クラス** | `BloodPactCard extends SkillCard` |
| **ファイル** | `lib/theomachia/cards/blood-pact.ts` |

#### 効果
- **自分のソウルを1減らす**
- **3枚ドロー**

#### 使用条件
- ソウルが1以上必要

#### 戦略メモ
- HP1点で3枚ドローは破格のアドバンテージ
- 残りHP1で使うと死亡リスクが一気に上がる
- 序盤に使ってキーカードを揃えるのが有効

### 祈り (Prayer)

> *回復＋ドロースキル。安定のユーティリティ。*

| 項目 | 値 |
|------|-----|
| **タイプ** | スキル |
| **回復** | 1 |
| **ドロー** | 1枚 |
| **色** | #F1C40F（黄） |
| **カテゴリ** | **オプション** |
| **クラス** | `PrayerCard extends SkillCard` |
| **ファイル** | `lib/theomachia/cards/prayer.ts` |

#### 効果
- **ソウルを1回復**（最大4まで）
- **1枚ドロー**

#### 戦略メモ
- 回復+ドローの安定カード
- 血の契約のデメリットを帳消しにできる
- 地味だが腐りにくい万能カード

## カード追加ガイド

新しいカードを追加するには以下の3ステップ:

### 1. カードクラスを作成

```typescript
// lib/theomachia/cards/sacred-fire.ts
import { SkillCard } from "./base";

export class SacredFireCard extends SkillCard {
  constructor() {
    super({
      id: "sacredFire",
      name: "聖火",
      description: "相手に1ダメージ。自分のソウルを1回復。",
      color: "#FF8800",
      optional: true,
    });
  }

  get effect() { return "sacredFire"; }
}
```

### 2. レジストリに登録

```typescript
// lib/theomachia/cards/index.ts
import { SacredFireCard } from "./sacred-fire";
// ALL_CARD_INSTANCES に追加:
new SacredFireCard(),
```

### 3. サーバー側に効果を実装

```typescript
// theomachia/src/card-effects.ts
sacredFire: {
  canPlay: () => true,
  play: (player, opponent, actions) => {
    actions.dealDamage(opponent, player, 1);
    if (player.souls < 4) {
      player.souls += 1;
      actions.logAction(`${player.name}のソウルが1回復`);
    }
  },
  needsTarget: () => "none",
},
```

### 4. カードイラストを追加（任意）

```
public/theomachia/cards/sacredFire.webp
```
