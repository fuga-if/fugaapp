# テオマキア マナレシオ分析

## マナレシオとは

カードのコスト（消費リソース）に対する効果量の比率。
テオマキアでは「マナ」は存在しないが、以下のリソースをコストとして計算する。

## コスト定義

| リソース | 基準値（1コスト） |
|---------|-----------------|
| プレイ権（行動回数） | 1プレイ = 1.0コスト |
| ソウル（HP自傷） | 1ソウル = 1.5コスト（HPは4しかない） |
| カードディスアド | 1枚捨て = 0.5コスト |
| 手札公開 | 情報漏洩 = 0.3コスト |

|------|------------------|

## 新カード評価テンプレート

### 手札交換カード（案）

**コスト:**
- 2プレイ権 = 2.0コスト

**効果:**
- 手札全交換 = ？

手札交換の価値は状況依存が大きい。
- 手札5枚 → 5枚交換: 各1枚の入れ替え ≒ 5 × 0.3 = 1.5（手札の質改善）
- ただし相手も良いカードを得る可能性: -1.0（リスク）
- 実質 0.5

**マナレシオ: 0.5 / 2.0 = 0.25**

→ **弱すぎる。** 2プレイ消費して手札交換はかなり厳しい。

### バランス案
| 案 | コスト | 追加効果 | 推定レシオ |
|----|-------|---------|-----------|
| A: 2プレイ+交換のみ | 2.0 | なし | 0.25 ← 弱い |
| B: 1プレイ+交換+1ドロー | 1.0 | +0.7 | 1.2 ← 適正 |
| C: 1プレイ+交換のみ | 1.0 | なし | 0.5 ← やや弱め |
| D: 2プレイ+交換+2ドロー | 2.0 | +1.4 | 0.95 ← ギリ適正 |

**推奨: 案B（1プレイ消費、手札全交換、その後1枚ドロー）**
理由: リスクが大きい（相手に良い手札を渡す）ので追加ドローで補う
